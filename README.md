# ApiProjectCamp

Kampus projesi için C# ile geliştirilmiş professional RESTful API uygulaması.

## 📋 Açıklama

Bu proje, .NET teknolojisi kullanılarak geliştirilmiş tam fonksiyonlu bir RESTful API'dir. CRUD operasyonları, veritabanı entegrasyonu ve modern API tasarım prensiplerini içerir.

## 🛠️ Teknolojiler

- **C#**
- **.NET Framework**
- **REST API**

## 🚀 Başlangıç

```bash
# Repository'yi klonlayın
git clone https://github.com/YunusEdizer/ApiProjectCamp.git

# Proje klasörüne gidin
cd ApiProjectCamp
```

## 📝 Özellikler

- RESTful API endpoints
- CRUD operasyonları
- Modern kod mimarisi
